# Certificado - Curso de Desenvolvimento de Aplicações Mobile com Android Studio


Este repositório contém o certificado de conclusão do curso Desenvolvendo Aplicações Mobile com Android Studio.

Sobre o Curso  

Neste curso, você aprenderá sobre o desenvolvimento de aplicações mobile e as plataformas disponíveis atualmente para criar aplicativos. O foco principal foi no Android Studio, uma das ferramentas mais utilizadas para o desenvolvimento de aplicativos Android.

Objetivos do Curso  

Compreender o conceito de desenvolvimento mobile e explorar as principais plataformas de desenvolvimento disponíveis.
Estudar o mercado de aplicações móveis e como se inserir nesse segmento.
Aprender a usar o Android Studio na prática, incluindo a criação de projetos, uso de layouts e requisições HTTP.
Desenvolver boas práticas de programação para garantir uma melhor experiência do usuário.
Conteúdo Programático

  Módulo 1 – Aplicativos mobile e plataformas de desenvolvimento

  Módulo 2 – Criando uma aplicação mobile e Activity

  Módulo 3 – Requisições HTTP
